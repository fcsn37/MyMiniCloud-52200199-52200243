MyMiniCloud

Xây dựng hệ thống Cloud thu nhỏ với 9 dịch vụ container
mô phỏng hạ tầng Cloud Platform hiện đại

Tổng quan hệ thống:

MyMiniCloud triển khai 9 container hoạt động đồng bộ trên Docker network cloud-net, bao gồm Web Server, Application Server, Database, Authentication, Object Storage, DNS, Monitoring (Prometheus & Grafana), và Reverse Proxy. Hệ thống đáp ứng đầy đủ yêu cầu đề bài và các yêu cầu mở rộng.

Phần 1

Web Frontend Server

Triển khai Nginx static site hosting với blog cá nhân. Bao gồm trang chủ, blog với 3 bài viết và cấu hình routing.
